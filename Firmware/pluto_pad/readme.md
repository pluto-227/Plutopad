# pluto_pad

![pluto_pad](imgur.com image replace me!)

This is a 9 key macropad that can be mapped to fit any need. It features a rotary encoder switch.

* Keyboard Maintainer: [Eli Mendez](https://github.com/pluto-227)
* Hardware Supported: *The PCBs, controllers supported*
* Hardware Availability: *Links to where you can find this hardware*

Make example for this keyboard (after setting up your build environment):

    make pluto_pad:default

Flashing example for this keyboard:

    make pluto_pad:default:flash
